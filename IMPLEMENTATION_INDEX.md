## üìû Common Questions

**Q: How do I upload a document?**
A: See [`QUICK_START_UPLOADS.md`](./QUICK_START_UPLOADS.md#how-to-use)

**Q: What file types are supported?**
A: See [`IMPLEMENTATION_GUIDE.md`](./IMPLEMENTATION_GUIDE.md#supported-file-types)

**Q: How large can files be?**
A: Max 10MB per file, up to 5 files per request

**Q: Where are files stored?**
A: Firebase Storage at `loans/{userId}/{loanRequestId}/`

**Q: Where is file metadata stored?**
A: Firestore `loans` collection, `documents` array

**Q: Can I customize the file limit?**
A: Yes! See [`IMPLEMENTATION_GUIDE.md`](./IMPLEMENTATION_GUIDE.md#-customization)

**Q: How do I configure security?**
